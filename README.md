# 🎵 Python MP3 Player for Mac
यह एक सरल MP3 प्लेयर है जिसे Python, Tkinter और Pygame का उपयोग करके बनाया गया है।

## 🚀 फीचर्स
- GUI (विंडो) आधारित इंटरफेस।
- Play, Pause, Stop और Volume कंट्रोल।
- GitHub Actions के जरिए ऑटोमेटेड CI टेस्टिंग।

## 🛠 इंस्टॉलेशन (Mac)
1. `brew install python-tk`
2. `pip install pygame`
3. `python3 music.py`
